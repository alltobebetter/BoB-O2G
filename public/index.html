OpenAI to Gemini API Proxy

这是一个 OpenAI API 兼容的 Gemini API 代理服务。

如何使用

本服务提供以下 API 端点：

  • /v1/chat/completions - 聊天完成
  • /v1/embeddings - 文本嵌入
  • /v1/models - 获取可用模型列表

配置示例

API Base URL:

https://[your-domain]/v1

环境变量设置:

OPENAI_API_BASE="https://[your-domain]/v1"
OPENAI_API_KEY="your-gemini-api-key"

注意事项

  • 需要使用 Gemini API Key 作为认证密钥
  • 支持流式响应
  • 支持图片和音频输入
  • 默认使用 gemini-1.5-pro-latest 模型
