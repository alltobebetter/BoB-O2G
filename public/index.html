AIBoB GO 代理服务

使用 Gemini API，享受 OpenAI 生态系统

快速开始

1. 获取 Gemini API Key

您可以直接通过 Gemini 的 APIKey 对 AIBoB Go 进行调用。

2. 可用 API 地址

基础地址：

https://go.aibob.click

OpenAI 格式基础地址：

https://go.aibob.click/v1

完整聊天接口地址：

https://go.aibob.click/v1/chat/completions

3. 配置环境变量

OPENAI_API_BASE="https://go.aibob.click/v1"
OPENAI_API_KEY="your-gemini-api-key"

代码示例 

from openai import OpenAI

client = OpenAI(
    base_url="https://go.aibob.click/v1",
    api_key="your-gemini-api-key"
)

response = client.chat.completions.create(
    model="gemini-1.5-pro-latest",
    messages=[
        {"role": "user", "content": "你好"}
    ]
)
